## 技术细节：Cytoscape DAG Bug 修复

**现象：** HTML 页面空白，无节点无连线

**排查过程：**
1. 检查 JS 初始化顺序 → `cytoscape()` 在 DOMContentLoaded 外直接调用
2. 检查 dagre 扩展注册 → `cytoscape.use()` 在 `cytoscape()` 之后
3. 检查 CDN 资源 → `cytoscape-dagre@2.5/dist/cytoscape-dagre.min.js` 返回 404

**根因：**
- npm 包 `cytoscape-dagre` 没有 `dist/` 目录，也没有 `.min.js` 文件
- 实际入口是包根目录的 `cytoscape-dagre.js`
- dagre 库加载失败 → `cytoscapeDagre` 未定义 → 扩展未注册 → layout 失败 → 节点无位置

**修复：**
```diff
- <script src="https://cdn.jsdelivr.net/npm/cytoscape-dagre@2.5/dist/cytoscape-dagre.min.js"></script>
+ <script src="https://cdn.jsdelivr.net/npm/cytoscape-dagre@2.5.0/cytoscape-dagre.js"></script>
```
